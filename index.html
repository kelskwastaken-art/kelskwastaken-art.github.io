Funcionalidades

🛡️

Teste de XSS

Detecta vulnerabilidades de Cross-Site Scripting em formulários e parâmetros URL.

🗃️

SQL Injection

Testa possíveis vulnerabilidades de injeção de SQL em aplicações web.

📁

Directory Traversal

Verifica falhas que permitem acesso a arquivos sensíveis do sistema.

🔍

Header Analysis

Analisa os cabeçalhos HTTP em busca de problemas de segurança.

🌐

Web Crawling

Coleta links e descobre estrutura do site automaticamente.

📊

Relatórios Detalhados

Gera relatórios completos em formato texto com os resultados dos testes.

Como Usar

Após baixar o arquivo, você pode executar a ferramenta de duas formas:

Modo Interativo (Recomendado para iniciantes):

WebPentestTool.exe

Modo Linha de Comando (Para usuários avançados):

WebPentestTool.exe https://exemplo.com -a

Opções disponíveis:

-a, --all     Executar todos os testes -x, --xss     Testar apenas XSS -s, --sql     Testar apenas SQL Injection -d, --dir     Testar apenas Directory Traversal -H, --headers Verificar apenas headers -c, --crawl  Apenas coletar links

Requisitos do Sistema

  • Memória RAM: 2GB ou mais
